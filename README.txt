דניאל שיווק — הוראות הפעלה

1. פתח את הקובץ index.html בלחיצה כפולה כדי לראות את האתר במחשב.
2. כדי לעדכן מספר טלפון ו-WhatsApp, פתח את config.js והחלף:
   whatsappNumber, phoneHref, phoneDisplay.
3. כדי להזין מחירים, החלף את price: 0 בכל מכונה ב-config.js.
4. לאחר העדכון, שמור ורענן את הדפדפן.
5. להעלאה בחינם: פתח חשבון ב-Netlify, גרור לשם את כל התיקייה או את קובץ ה-ZIP.

הערה: כרגע הגלריה משתמשת באיורים ולא בתמונות אמיתיות. מומלץ להחליף בהמשך.
